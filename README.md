# Movie Review Aggregation Database

A Rotten Tomatoes-inspired relational database designed to aggregate movie reviews, critic ratings, and audience feedback.

This project implements **POPCORN_INDEX**, a fully normalised MySQL database that supports movie review aggregation and analytics. The system models films, critics, reviews, scores, genres, franchises, actors, and user interactions while maintaining data integrity through constraints, triggers, and stored functions.

Key features include:
- 🎬 Movie and TV feature management
- 📝 Critic review aggregation
- 🍅 Hitmeter and Certified Fresh calculations
- ⭐ Weighted critic scoring
- 📈 Trending film analysis
- 🔒 Database-level integrity enforcement
- 🗄️ BCNF-compliant relational design

**TECHNOLOGIES -**
- MySQL
- SQL
- ER Modelling
- Database Normalisation
- Triggers & Stored Functions
